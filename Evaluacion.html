Selector Global

Se utiliza para seleccionar todos los elementos de la página.

El selector universal se indica mediante un asterisco (*). A pesar de su sencillez, no se utiliza habitualmente, ya que es difícil que un mismo estilo se pueda aplicar a todos los elementos de una página.

No obstante, sí que se suele combinar con otros selectores y además, forma parte de algunos hacks muy utilizados, como se verá más adelante.

Es el nivel más bajo de la jerarquía. Selecciona absolutamente todos los elementos dentro del documento HTML (desde el y hasta el último enlace o párrafo).

Selector de Clase

Es la herramienta más versátil. Permite agrupar diferentes elementos bajo un mismo estilo. Su gran ventaja es la reutilización..

Regla: Un elemento HTML puede tener múltiples clases (separadas por espacios) y una misma clase puede usarse en infinitos elementos.

Selector ID

Es un selector de máxima precisión. Se utiliza para identificar un elemento que es único en toda la estructura de la página..

Regla de Oro: No puedes repetir el mismo ID en dos elementos distintos del mismo documento. Si lo haces, el HTML será inválido y podrías tener problemas con JavaScript.

Imagen

Se utiliza para insertar imágenes en una página web. No incrusta la imagen directamente en el archivo, sino que crea un espacio reservado y le indica al navegador dónde encontrarla para cargarla
